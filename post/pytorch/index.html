PyTorch常用

22 Dec 2019

import torch.nn as nn （对象）

  • nn.Conv2d(in_channels, out_channels, kernel_size, stride=1,padding=0)
  • nn.Dropout2d()
  • nn.Linear(in_features, out_features)
  • nn.ReLU()

import torch.nn.functional as F (函数)

  • F.max_pool2d(input_tensor, kernel_size)
  • F.relu(input_tensor)
  • F.dropout(input_tensor)
  • F.log_softmax(input_tensor, dim=None)
  • F.nll_loss(input_tensor, output_tensor) [negative log likelihood loss]

import torch.optimal as optimal (对象)

  • optimal.SGD(params, lr=0.01, momentum=0.9)

  • optimal.Adam(params, le=0.001)

    Note: params=model.parameters()

其他

  • reshape: tensor.view(shape)
  • reshape: tensor.view(tensor)
  • 比较，相同返回1，不同返回0: tensor.eq(tensor)
  • 求和：tensor.sum()
  • 求最大值：tensor.max()
  • 获取tensor的实值：tensor.item()

查询

  • cuda是否可用：torch.cuda.is_available()
  • 返回gpu数量：torch.cuda.device_count()
  • 返回gpu名字：torch.cuda.get_device_name(0)
  • 返回当前设备索引：torch.cuda.current_device()
