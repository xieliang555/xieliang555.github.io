常用命令行

  • git init 初始化本地仓库
  • git remote add origin RemoteUrl 将本地仓库与远程仓库建立关联
  • git remote rm origin 解除关联(不会删除仓库中的内容)
  • git remote -v 查看远程仓库
  • git status
  • git add .
  • git commit -m “mark info”
  • git pull origin master –allow-unrelated-histories 拒绝合并无关历史
  • git push origin master
  • 当两端origin端和master端同时改变，会产生冲突

清空仓库（同时保留仓库）

  • git pull origin master (保持与origin仓库同步)
  • git rm * -r (删除本地所有文件)
  • git add . （提交）
  • git commit -m “delete all”
  • git push origin master
