虚拟环境

  • 创建虚拟环境： conda create -n EnvName python=3.6
  • 删除虚拟环境：conda remove -n EnvName —-all
  • 重命名环境：
    • conda create -n NewEnvName –clone oldEnvName
    • conda remove -n OldEnvName –all
  • 查看虚拟环境：conda env list

安装

  • 三方库的两种安装方式
    • 利用pip或conda在线安装
    • 通过wheel离线安装
  • 开源代码的两种安装方式。无论哪一种安装方式，都最好先创建虚拟环境，因为安装时需要安装依赖包（库）
    • 利用pip或者conda通过wheel安装
    • 从source（源代码）安装。如果是项目最好从source安装，1)可以看到源代码。2) 可以在项目基础上更方便修改（如spinningUp，cs294HW 避免使用命令行而直接使用sublime给定参数）
  • 更新pip: pip install —upgrade pip
  • 忽略已安装的库： pip install —ignore-installed PkgName
  • 安装指定版本库：conda install pkg=1.0.1 / pip install Pkg==1.0.4
  • 更新pkg： pip install —upgrade pkg
  • 通过python脚本安装包： pip install -e . 执行当前目录下的setup.py脚本
  • 通过txt文件安装包：pip install -r “requirements.txt”
    • gym==0.10.5
    • mujoco-py >=1.50.1.56
    • tensorflow
    • numpy
    • seaborn
  • 临时更改镜像源：pip install -i Pkg

查询

  • 查询已安装的库：conda list / pip list
  • 查询python位置：which python
  • 查询python命令：python -h
  • 查询conda命令：conda -h

镜像源

  • pip源
    • 在home目录新建文件夹pip

    • 在pip文件夹下新建文件pip.ini，并输入

      [global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
[install]
trusted-host=mirrors.aliyun.com
  • conda源 （执行以下命令行会在home目录自动创建.condarc文件）
    • conda config —add channels
    • conda config —add channels
    • conda config —set show_channel_urls
  • 查看conda源： conda config –show
