虚拟环境

  • 创建虚拟环境： conda create -n EnvName python=3.6 (考虑是否需要将kernel装入jupyter notebook)
  • 删除虚拟环境：conda remove -n EnvName —-all
  • 重命名环境：
    • conda create -n NewEnvName –clone oldEnvName
    • conda remove -n OldEnvName –all
  • 查看虚拟环境：conda env list

安装

  • 三方库的两种安装方式
    • 利用pip或conda在线安装
    • 通过wheel离线安装
  • 开源代码的两种安装方式。无论哪一种安装方式，都最好先创建虚拟环境，因为安装时需要安装依赖包（库）
    • 利用pip或者conda通过wheel安装
    • 从source（源代码）安装。实际是从安装包中安装，安装完后可以删除
  • 更新pip: pip install —upgrade pip
  • 忽略已安装的库： pip install —ignore-installed PkgName
  • 安装指定版本库：conda install pkg=1.0.1 / pip install Pkg==1.0.4
  • 更新pkg： pip install —upgrade pkg
  • 通过python脚本安装包： pip install -e . 执行当前目录下的setup.py脚本
  • 通过txt文件安装包：pip install -r “requirements.txt”
    • gym==0.10.5
    • mujoco-py >=1.50.1.56
    • tensorflow
    • numpy
    • seaborn
  • 临时更改镜像源：pip install -i Pkg
  • conda install 出现 Solving environment: Done时，使用conda clean –all清除无用包。 可能是之前在conda下载包时被终止造成无向缓存。

查询

  • 查询已安装的库：conda list / pip list
  • 查询python位置：which python
  • 查询python命令：python -h
  • 查询conda命令：conda -h

镜像源

  • pip源
    • 在home目录新建文件夹.pip

    • 在pip文件夹下新建文件pip.ini，并输入

      [global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
[install]
trusted-host=mirrors.aliyun.com
  • conda源 （执行以下命令行会在home目录自动创建.condarc文件）
    • conda config —-add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
    • conda config —-set show_channel_urls
  • 查看conda源： conda config –show
  • 重启

Jupyter Notebook

  • 添加Kernel
    • 方法0 新建+导入：
      • conda –version 检测版本是否在4.1.0以上，不在则更新：conda update conda
      • conda list 检测是否已安装库nb_conda_kernels ，没有则安装：conda install nb_conda_kernels
      • 新建环境，并将环境导入jupyter notebook: conda create -n env_name python=3.6 ipykernel
    • 方法1: https://blog.csdn.net/wj1066/article/details/72891667
    • 方法2 导入:
      • 命令行进入该环境
      • conda install -n EnvName ipykernel
      • python -m ipykernel install –user
      • 刷新即可，不需要重启jupyter
  • 查询kernel：
    • jupyter kernelspec list （最多显示两个可用kernel，删除一个后可显示其他的kernel）
    • conda虚拟环境的kernel不会显示

  • 删除kernel ：

    • jupyter kernelspec uninstall kernel-name
    • 若查询kernel未显示，直接删除conda环境也可以从jupyter notebook中删除kernel

  • 添加目录及代码补全,高亮显示等

    • pip install jupyter_contrib_nbextensions
    • jupyter contrib nbextension install –user
    • 重启
    • 选择 Nbextensions，在笔记本菜单栏中点击table of contents图标会出现content
    • 勾选 Table of Contents
    • 勾选 Hinterland
    • 勾选 Highlight selected word
    • 勾选 Autopep8,自动整理代码（需安装）

  • 显示md文件

    • 安装pip install https://github.com/aaren/notedown/tarball/master (或者pip install notedown) 【会乱码】
    • 生成配置文件jupyter_notebook_config.py（如果没有的话）jupyter notebook –generate-config
    • 配置文件路径 ~/.jupyter/jupyter_notebook_config.py
    • 打开配置文件，在末尾添加c.NotebookApp.contents_manager_class = ‘notedown.NotedownContentsManager‘
    • 或者命令行：jupyter notebook –NotebookApp.contents_manager_class=‘notedown.NotedownContentsManager’

  • 运行计时

    • pip install jupyter_contrib_nbextensions
    • jupyter contrib nbextension install –user
    • jupyter nbextension enable execute_time/ExecuteTime
