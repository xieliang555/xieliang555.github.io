Win10+Ubuntu

25 Dec 2019

安装

  • 制作U盘启动盘：查看系统的启动方式和分区模式

    • UEFI启动+GTP（guid）分区
    • 传统Legacy BIOS启动+MBR分区
    • 可通过第三方分区软件将GPT和MBR分区无损转换，然后在在bios中将legacy启动设为UEFI启动

  • 创建硬盘空白分区：我的电脑-管理-磁盘管理，制作安装ubuntu的空白分区

  • 关闭win10的快速启动和安全启动

    • 关闭快速启动：电源与睡眠 -> 其他电源设置 -> 选择电源按钮的功能 -> 更改当前不可用的设置 ->取消快速启动 （也可在bios中更改）
    • 关闭安全启动：开机按F2或DEL， 进入BIOS模式 -> 选择security -> secure boot 设置成disable -> 在boot选项查看快速启动是否关闭 （为了通过u盘启动）[无线键盘可能无法进入bios]
    • 安装ubuntu：开机按F12，选项U盘启动方式 [或者在bios中设置启动顺序]
    • 检测到这台电脑已经安装了windows boot manager后，选择其他选项
    • 分区：选择空闲的空间，继续分区，swap+/
      • Swap交换空间：内存大小的1～1.5倍。逻辑分区+空间起始位置
      • 其余的分给 / 。 逻辑分区+空间起始位置+ext4日志文件系统
    • 将启动引导器安装在和/一样的位置中

  • 如果需要重新安装，不要直接从window下格式化（可能会导致无法启动windoes），重新从u盘启动，进入到分区界面，将之前Ubuntu分区的通过减号“-”去除，然后通过加号”+“重新添加分区，要卸载ubuntu也是通过这种方法。

    注意：

    • 先安装windows，后安装ubuntu，由ubuntu引导windows启动（不能在window下直接格式化ubuntu系统）
    • ubuntu和windows可以同时安装在同一块硬盘中
    • 在ubuntu中无法访问windows磁盘，可能是windows处于休眠状态，先打开windows然后再启动ubuntu（windows中的快速启动必须关闭）
