文件结构

  • module: 模块，具有独立功能，函数或类构成的module.py文件，通过import导入，通过module.func() 调用
  • Package: 相关模块的集合，结构：
    • package
      • init.py
      • – module1.py
      • – module2.py
  • Library:： 相关模块的集合
  • 包，库，模块三个概念都是指模块，只是个体和集合的区别
  • 依赖：包（库）与包（库）之间的引用关系

常用函数：

  • 字符串

    • str.repalce(char, char)
    • Str.split(char)

  • 文件

    • open(file_path, ‘r’, encoding=‘utf-8’)
    • f.read()

  • 基本数据结构

    • dict()
    • dict.items()
    • dict.keys()
    • dict.values()
    • dict.get()
    • list.insert()
    • list反转：[::-1]
    • List.extend()

  • 路径

    • os.getcwd() 获取当前工作路径
    • os.path.join(root, relative_path) 获取指定路径

  • 其他

    • enumerate
    • zip
    • copy ref
    • iter()
    • next()
    • type()

Python脚本执行方式（查找文件的路径不一样）： 1. python file.py 运行当前路径下的file.py文件 2. python -m file.py 将已安装的模块当脚本运行
