文件结构

  • module: 模块，具有独立功能，函数或类构成的module.py文件
  • Package: 相关模块的集合，结构：
    • package
      • init.py
      • – module1.py
      • – module2.py
  • Library:： 相关模块的集合
  • 如：Library： PyTorch； Package： torchvision，torchtext； Module： torchtext.data
  • 依赖：包（库）与包（库）之间的引用关系

  • 编码

    • 当前文件的编码格式：python3 的每一个头文件都会有 # -- encoding==utf-8 -- 它的实际作用是将当前py文件的编码格式设置为utf-8，如果没有这一行注释代码， 那么在执行当前的.py文件时候，只要该.py文件中有中文汉字就会ascii报错
    • 当前系统使用的默认编码：在python27版本中，当前环境的默认编码为ascii，每当你创建一个字符串str对象时，它的编码格式为默认的ascii格式，如果在执行字符串操作时候报了相关的ascii 错误，可以在该文件中添加以下语句，使当前系统的默认编码修改为utf-8，python3之后的版本不需要，已经默认字符编码是utf-8

    • 标准输出默认编码：print默认编码方式是ascii，在print一个中文字符串时候有时候也会报ascii的编码错误，可以通过设置改为uft-8

      import sys
import codecs
sys.stdout = codecs.getwriter("utf-8")(sys.stdout.detach())
