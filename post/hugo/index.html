目录结构

  • blogName
    • content：存储着md（markdown）源文件
    • Public：编译后生成的html文件，将public文件夹push到github即可通过互联网访问
    • Static：存储md文件中的图片
    • Themes：存储各种主题
    • config.toml：配置文件（一般下载的theme中会提供exampleSite的配置，每次换主题需要换配置文件）

安装及部署

  • brew install hugo
  • 创建博客：Hugo new site blogName，生成目录结构
  • 下载主题： themes.gohugo.io，放入themes目录中，可通过更改文件夹名称改变theme名称
  • 在config.toml中可以配置默认theme主题，模版见参考url
  • 本地打开blog：hugo server -D (可通过 –theme=themeName指定主题)
  • hugo （可通过–theme=themeName），生成public目录（存放着html文件，用于push到远程仓库）
  • 进入public目录，创建本地仓库（git init）
  • 创建github远程仓库：必须以xieliang555.github.io命名
  • 本地仓库关联到远程仓库(git remote add origin)，并将本地的public目录push到远程仓库
  • 互联网访问xieliang555.github.io
  • 注意：md文件头中的配置信息中draft为true时，执行hugo后不会输出到public目录中

写博客

  • 新建文章：hugo new post/blog.md（会在content目录下生成该文件）
  • 写文章：进入blog.md目录，open blog.md

配置MathJax

  • 配置MathJax使md文件支持MarkDown（Latex）公式编辑

  • 在博客根目录下的/layout/partials路径下新建文件mathjax_support.html，并添加以下代码

         <script type="text/javascript" async
      src="https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML">
      MathJax.Hub.Config({
      tex2jax: {
        inlineMath: [['$','$'], ['\\(','\\)']],
        displayMath: [['$$','$$']],
        processEscapes: true,
        processEnvironments: true,
        skipTags: ['script', 'noscript', 'style', 'textarea', 'pre'],
        TeX: { equationNumbers: { autoNumber: "AMS" },
             extensions: ["AMSmath.js", "AMSsymbols.js"] }
      }
      });
      MathJax.Hub.Queue(function() {
        // Fix <code> tags after MathJax finishes running. This is a
        // hack to overcome a shortcoming of Markdown. Discussion at
        // https://github.com/mojombo/jekyll/issues/199
        var all = MathJax.Hub.getAllJax(), i;
        for(i = 0; i < all.length; i += 1) {
            all[i].SourceElement().parentNode.className += ' has-jax';
        }
      });

      MathJax.Hub.Config({
      // Autonumbering by mathjax
      TeX: { equationNumbers: { autoNumber: "AMS" } }
      });
     </script>

  • 在该目录下的文件header.html（没有则新建）的前添加以下代码：

        {{ partial "mathjax_support.html" . }}
